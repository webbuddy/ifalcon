TouchDown Web Server Beta Info

1. Setup.
Run WebServer.exe and note the LAN IP address this should only work on your internal network and the devices connected to it . Open web browser on the device of your choice and enter the IP address and the default profile (index.htm found in Data\Docs) will be loaded. Test as required.

2. Changing / Creating Commands.
To create your own profile/interface your going to know how to use some basic html even if its just in a text editor - open the default profile with your editor of preference and find the weblinks that start <a href" The command to send a key uses a weblink but in the form of a command - it must always start with the word function this is how the server application knows that your sending a command instead of requesting a webpage.

The link must also have a target set of bottomFrame which tells your browser to open the resulting return information in the bottom frame of the page (leaving the current page unaffected) this has been implemented as the browser requires this return info or it will crash/timeout, so the return info is just your command echoed back - (temp.htm found in Data\Docs and is always required for this so don't delete it).

3. Examples

Examples : <a href='function macro("hello");' target="bottomFrame">
Sends the text hello

Examples : <a href='function macro("{SPACE}");' target="bottomFrame">
Sends the space key

Examples : <a href='function macro("{SPACE}");macro("{UP}");macro("{DOWN}");' target="bottomFrame">
Sends multiple commands space - up - down

As you can see in these example the actual key press/presses are captured in brackets and quotation marks ("keys to send") normal letters or strings of text are sent as seen in example 1 and a default 30milliseconds is inserted between each keystroke. The 2nd example shows a special key in this case the space key. The special keys are listed in a table below and must always be enclosed in Parentheses{}.

4. Table of Special Keys
TouchDown uses reserved symbols to perform certain additional actions and these are listed below along with other special keys - if its not listed here then no special treatment is required

'!' - This tells TouchDown to send an ALT keystroke, therefore <a href='function macro("This is text!a");' target="bottomFrame"> would send the keys This is text and then press ALT+a.

N.B. Some programs are very choosy about capital letters and ALT keys if in doubt, use lowercase!

'+' - This tells TouchDown to send a SHIFT keystroke, therefore <a href='function macro("Hell+o");' target="bottomFrame"> would send the text HellO.<a href='function macro("!+a");' target="bottomFrame"> would send ALT+SHIFT+a.

'^' - This tells TouchDown to send a CONTROL keystroke, therefore <a href='function macro("^!a");' target="bottomFrame"> would send CTRL+ALT+a.

N.B. Some programs are very choosy about capital letters and CTRL keys if in doubt, use lowercase!

5. The Pause Function
Sometimes when stringing multiple macros to a single control you may require to insert a short delay between key presses to do so use the pause(number in milliseconds) command - please note no quotation marks are required within the pause command

Examples : <a href='function pause(1000);' target="bottomFrame">
Inserts a 1 second pause as 1000ms = 1 sec

Examples : <a href='function macro("{SPACE}");pause(300);macro("{ENTER}")' target="bottomFrame">
Sends the space key, then pauses 300ms, then sends the enter key
